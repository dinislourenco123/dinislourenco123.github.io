Glossário

HTML
   HTML (HyperText Markup Language) é a linguagem usada para estruturar páginas da Internet. Serve para organizar o conteúdo, como textos, imagens, vídeos e links através de etiquetas (tags).

CSS
   CSS (Cascading Style Sheets) é a linguagem usada para definir o aspecto visual das páginas da Internet criadas em HTML.

MySQL
   MySQL é um sistema de gestão de bases de dados (SGBD) que usa a linguagem SQL para guardar, organizar e consultar informação.

WordPress
   WordPress é um sistema de gestão de conteúdos (CMS) que permite criar e gerir sites sem precisares de programar.
